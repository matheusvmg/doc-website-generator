# Getting Started

This section covers everything you need to get up and running with our project.

## Prerequisites

Before you begin, make sure you have the following installed:

- **Node.js** 18.0 or later
- **npm** 9.0 or later (or yarn/pnpm)
- **Git** 2.30 or later

> [!IMPORTANT]
> Node.js 18 is the minimum required version. Earlier versions are not supported.

## Overview

The setup process involves three main steps:

1. Install the package
2. Configure your project
3. Start the development server

Continue to the [Installation](/docs/getting-started/installation) guide to get started.
