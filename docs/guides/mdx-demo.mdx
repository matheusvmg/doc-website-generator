# MDX Components

Welcome to the MDX demo! You can use React components directly in your markdown.

## Callouts

Use the `<Callout>` component to highlight important information.

<Callout type="note">
  This is a standard note. It uses the `Callout` component.
</Callout>

<Callout type="warning">
  Be careful! This is a warning callout.
</Callout>

<Callout type="tip">
  **Pro Tip**: You can use markdown inside callouts too!
</Callout>

## Regular Markdown

Regular GFM features still work perfectly:

- **Bold** and *Italic*
- [Links](/docs)
- `Inline code`
- Blocks of code:

```typescript
function hello() {
  console.log("Hello from MDX!");
}
```

| Feature | Support |
| ------- | ------- |
| MDX     | Yes     |
| GFM     | Yes     |
| Shiki   | Yes     |
